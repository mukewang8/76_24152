<h3>课程介绍：</h3>
<p>通过学习，掌握三维图形学的基本理论，三维数学知识，并根据数理进行实践，纯软件编码方式（不调用OpenGL,DX，或者相关图形库）实现一个简单的三维场景，巩固三维理论，为后续课程打下基础。</p>
<p>学习图形学理论知识，并以纯软件的方式，不调用第三方(OpenGL，D3D)三维API,纯软件方式实现三维场景的绘制，第三人称摄像机的控制，从而深刻的理解三维理论。</p>
